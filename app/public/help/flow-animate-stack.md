# Animate Stack

Each frame appears on a desk one by one, randomly rotated. Generic — use after `overlay-polaroid-frame` for a polaroid stack. `isolateSubject=true` runs InSPyReNet on every source image and bakes the matte into alpha, so frames composite as cut-out silhouettes rather than rectangles (requires #mdl; N sources = N inferences).
